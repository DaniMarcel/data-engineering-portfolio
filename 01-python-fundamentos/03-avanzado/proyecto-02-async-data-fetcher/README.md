# ⚡ Async Data Fetcher

Programación asíncrona con `asyncio` y `aiohttp` para descarga paralela de datos.

## Quick Start

```bash
pip install -r requirements.txt
python src/async_fetcher.py
```

## Conceptos

- async/await syntax
- aiohttp for async HTTP
- asyncio.gather for parallel execution
- Semaphore for rate limiting
- Async pipelines

---

Mucho más rápido que peticiones síncronas para múltiples URLs.
