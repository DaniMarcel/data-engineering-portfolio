# ⚡ Introducción a PySpark

Apache Spark con Python para procesamiento distribuido.

## Instalación

```bash
pip install pyspark
```

## Quick Start

```bash
python src/intro_spark.py
```

## Conceptos

- SparkSession
- DataFrames distribuidos
- Transformaciones (lazy)
- Acciones (eager)
- GroupBy y agregaciones
- Spark SQL

## Features

- createDataFrame desde Python
- read.csv con inferSchema
- Transformaciones con withColumn
- GroupBy con múltiples agregaciones
- SQL queries sobre DataFrames
- Estadísticas descriptivas

---

Spark procesa Big Data en memoria de forma distribuida.
